Swing Property Editor

This project is a Swing UI for editing properties in a tree-like format, using a series of metadata elements for determining datatypes, etcetera.

For example, given the sample properties file: 

		world.continent.America.USA.Montana=Montana
		world.continent.America=America
		world.continent.Europe=Europe
		world.continent.America.Mexico=Mexico
		world.continent.America.Mexico.Morelos=Morelos
		world.continent.America.USA=United States
		world.continent.America.Canada=Canada, eh?
		world.continent.Europe.Spain=Spain
		world.continent.America.USA.Wisconsin=Wisconsin
		world.continent.America.USA.California=California

The property editor uses a metadata xml descriptor file (see sample) for giving extra information during the editing operations.

In addition, the property editor can be used as a simple control that you can pass a Properties object to. Under this context, the Property editor uses an editor factories to display the most appropriate editor for the value object even if it is not a simple string.

Currently implemented Editors include:

  • String Editor (standard)
  • Boolean value editor
  • Color Editor
  • Font Editor
